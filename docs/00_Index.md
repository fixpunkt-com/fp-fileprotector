# Zugriffsschutz für File Storages
Mit *fp-fileprotector* kannst du den Zugriff auf File Storages einschränken und an Regeln knüpfen. Dabei kannst du den Zugriff wie folgt einschränken:
* Ob ein:e Benutzer:in im Frontend eingeloggt ist.
  * ...und Mitglied einer Benutzer:innen-Gruppe ist.
  * ...und ein spezifische:r Benutzer:in ist.
* Ob ein:e Benutzer:in im Backend angemeldet ist.

## Schützen eines Storages
Wie du einen FileStorage schützt erfährst du im Kapitel [Einen Storage schützen](01_Einen Storage schützen.md).

Wie du Zugriffsregeln anlegst erfährst du im Kapitel [Zugriffsregeln](02_Zugriffsregeln.md).
